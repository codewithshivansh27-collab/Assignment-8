# Weather Dashboard

A simple weather dashboard that fetches weather for 3 cities simultaneously using `Promise.all` and the Open-Meteo API.

## Features

- Fetches weather for 3 cities in parallel
- Displays city name, temperature, condition, and weather emoji
- Loading spinner while data is being fetched
- Error handling if API request fails
- Clean card-based UI

## Files

- `index.html`
- `style.css`
- `script.js`

## Run

Just open `index.html` in your browser.
